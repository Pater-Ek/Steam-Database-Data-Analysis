# Steam Game Success Prediction (ML Project)

This project focuses on analyzing the Steam games dataset and building a machine learning pipeline.

The goal of this project was to apply data analysis, feature engineering, and machine learning techniques to a real-world dataset.

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- PyTorch
- MATLAB

## Dataset

Dataset: https://www.kaggle.com/datasets/nikdavis/steam-store-games?resource=download
